# Development

This part is for developers. If you want to help to the project, you can join the discord server there.

# Language(s)

All the project will use Typescript which is compatible and very similar to Javascript. The final goal is to be able to have a working system running with NodeJS.

# Project structure

The project is separated in 3 parts:

  • server: a web app which will permit modpack's creator and server's owner to manage modpacks and launchers
  • client: a Node.JS library which permit to download modpacks since the configuration downloaded from the server
  • documentation: this is this website which will need to be maintened to stay updated and translated in multiple languages.
